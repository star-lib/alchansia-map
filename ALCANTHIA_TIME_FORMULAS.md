# Alcanthia Time Formulas

## 개요

[alcanthia_index.js](/C:/Users/bigdr/Documents/codex/proj2/alcanthia_index.js) 안 실제 게임 로직을 기준으로, 양조 / 강화 / 제작 / 각인 계열 작업 시간 계산 방식을 정리한 문서입니다.

이 게임은 작업 종류마다 완전히 다른 시간표를 두는 방식이 아니라, 대부분 공통 계산식을 사용합니다.

핵심 포인트:

- 각 아이템은 `brewDuration`이라는 기본 시간을 가짐
- 강화 수치가 붙으면 시간이 배로 증가함
- 실제 작업 시작 시에는 재료 2개 중 더 오래 걸리는 쪽을 기준으로 계산함
- 일부 스킬/효과로 최종 시간이 감소할 수 있음

## 1. 기본 시간

게임 내부 기본 함수:

```js
function XP(i){ return st[i].brewDuration }
```

의미:

- 어떤 아이템의 기본 작업 시간은 `brewDuration` 값입니다.
- 이름은 `brewDuration`이지만 실제로는 양조 전용이 아니라 강화/제작/각인에도 공통 기반 시간으로 사용됩니다.

예시:

- 씨앗 일부: `15초`, `60초`
- 작물 일부: `15초`, `30초`, `60초`, `120초`, `180초`
- 가마솥:
  - 낡은 가마솥: `30초`
  - 구리 가마솥: `60초`
  - 은 가마솥: `120초`
  - 금 가마솥: `240초`

## 2. 강화 반영 시간

게임 내부 함수:

```js
function Ll(item){
  return XP(item.itemCode) * Math.pow(2, item.enhancement)
}
```

의미:

- 강화 수치가 1 오를 때마다 시간이 2배가 됩니다.

공식:

```text
강화 반영 시간 = 기본시간 × 2^(강화수치)
```

예시:

- 기본시간 `15초` 아이템
  - `+0` = `15초`
  - `+1` = `30초`
  - `+2` = `60초`
  - `+3` = `120초`

- 기본시간 `60초` 아이템
  - `+0` = `60초`
  - `+1` = `120초`
  - `+2` = `240초`
  - `+3` = `480초`

## 3. 실제 작업 시간

게임 내부 함수:

```js
function Gr(a, b, cauldronEnhancement, skills){
  const base = Math.max(Ll(a), Ll(b))
  const flameMastery = skills.flame_mastery ?? 0
  const reduction = flameMastery > 0 ? Ds(.01, flameMastery, cauldronEnhancement + 1) : 0
  return Math.round(base * (1 - reduction))
}
```

의미:

- 재료 2개를 넣는 작업은 둘 중 더 오래 걸리는 재료를 기준으로 시간 계산
- 그 뒤 일부 스킬 보정이 들어감

공식:

```text
실제 작업 시간 = round( max(재료1 강화반영시간, 재료2 강화반영시간) × (1 - 시간감소보정) )
```

즉:

1. 각 재료의 기본시간을 가져옴
2. 강화 수치만큼 `2^강화수치`를 곱함
3. 둘 중 큰 값을 선택
4. 시간 감소 효과가 있으면 적용
5. 반올림해서 최종 시간 결정

## 4. 어떤 작업에 적용되나

이 공식을 공통으로 쓰는 작업:

- 양조 `potion`
- 제작 `craft`
- 강화 `enhancement`
- 각인 `engrave`
- 유틸 조합 `utility`
- 실패/무효 조합 `invalid`도 내부 대기시간은 같은 방식으로 생성

즉, 양조만 특별 계산식을 쓰는 구조가 아닙니다.

## 5. 추가 시간 감소 요소

### flame_mastery

`Gr(...)` 내부에서 직접 적용됩니다.

- `flame_mastery`가 높을수록 최종 시간이 감소
- 정확한 감소율 계산은 `Ds(.01, flame_mastery, cauldronEnhancement + 1)` 쪽 보조 함수에 들어 있음
- 따라서 스킬 레벨과 가마솥 강화 상태에 따라 감소량이 달라질 수 있음

### fire_potion

강화 시작 함수 쪽에서 추가 감소가 들어갑니다.

로직 요약:

```js
r = Math.round(r * (1 - (enhancement + 1) * 0.1))
```

의미:

- `fire_potion` 효과가 남아 있으면 추가 시간 감소
- 포션 강화 수치가 높을수록 감소폭이 커짐

예시:

- `fire_potion +0`이면 `10%` 감소
- `fire_potion +1`이면 `20%` 감소
- `fire_potion +2`이면 `30%` 감소

이 보정은 현재 확인한 로직상 강화 쪽 시작 처리에서 직접 적용됩니다.

## 6. 식물 성장/생산 시간은 별도

텃밭 식물은 `brewDuration`이 아니라 별도 테이블을 씁니다.

관련 필드:

- `growTime`: 성장 완료까지 걸리는 시간
- `waterInterval`: 물 필요 주기
- `produces[].interval`: 생산 주기
- `produces[].max`: 최대 저장량

예시:

- `red_flower`
  - 성장 시간 `10초`
  - 물 주기 `20초`
  - 생산 주기 `15초`

- `blue_moss`
  - 성장 시간 `25초`
  - 생산 주기 `40초`
  - 최대 저장 `2`

- `poison_flower`
  - 성장 시간 `25초`
  - 물 주기 `25초`
  - 생산 주기 `10분`

즉, 텃밭은 “작업 대기 시간”과 “식물 성장/생산 시간”이 분리되어 있습니다.

## 7. 이해하기 쉬운 요약

가장 단순하게 보면:

```text
아이템 시간 = 기본시간 × 2^강화수치
작업 시간 = 재료 2개 중 더 긴 시간
최종 시간 = 여기에 스킬/효과 감소 적용
```

예를 들어:

- 재료 A: 기본 `30초`, 강화 `+2` -> `120초`
- 재료 B: 기본 `60초`, 강화 `+0` -> `60초`

그러면 작업 기준 시간은:

```text
max(120초, 60초) = 120초
```

여기에 `flame_mastery`나 `fire_potion` 같은 감소 효과가 있으면 최종 시간이 더 짧아집니다.

## 8. 결론

- 양조 / 강화 / 제작은 공통 시간 계산 공식을 사용합니다.
- 중심이 되는 값은 각 아이템의 `brewDuration`입니다.
- 강화 수치가 붙을수록 시간은 `2배`, `4배`, `8배` 식으로 증가합니다.
- 실제 작업 시간은 재료 둘 중 더 긴 쪽 기준으로 정해집니다.
- 식물 성장/생산만 별도 `growTime`, `interval` 체계를 사용합니다.

## 9. 빠른 계산 절차

문서만 보고 시간을 계산하려면 아래 순서로 보면 됩니다.

1. 재료 2개의 `기본시간`을 아래 표에서 찾습니다.
2. 각 재료에 강화 수치가 있으면 `기본시간 × 2^강화수치`를 계산합니다.
3. 둘 중 큰 값을 고릅니다.
4. `flame_mastery`, `fire_potion` 같은 감소 효과가 있으면 마지막에 적용합니다.

가장 자주 쓰는 계산 형태:

```text
재료 A 최종시간 = A 기본시간 × 2^A강화
재료 B 최종시간 = B 기본시간 × 2^B강화
작업 기준시간 = max(A 최종시간, B 최종시간)
최종 작업시간 = 기준시간 × (시간감소 보정 적용)
```

예시 1:

- `약초(+0)` + `푸른이끼(+0)`
- 약초 기본시간 `15초`
- 푸른이끼 기본시간 `30초`
- 결과: `max(15, 30) = 30초`

예시 2:

- `붉은꽃잎(+2)` + `붉은꽃잎(+0)`
- 붉은꽃잎 기본시간 `30초`
- 계산: `30×4 = 120초`, `30×1 = 30초`
- 결과: `max(120, 30) = 120초`

예시 3:

- `은광석(+1)` + `진흙(+0)`
- 은광석 기본시간 `60초`
- 진흙 기본시간 `30초`
- 계산: `60×2 = 120초`, `30×1 = 30초`
- 결과: `120초`

### 표 읽는 법

- `성장시간`: 심은 뒤 첫 활성화까지 걸리는 시간
- `물 주기`: 다시 물이 필요한 간격
- `생산물 주기`: 생산물이 1개 차오르는 간격
- `최대 저장`: 한 번에 쌓일 수 있는 최대 수량

## 12. 실전 해석 팁

### 포션 조합 시간

포션 레시피를 만들 때는 결과 포션 시간이 아니라, 넣은 재료 2개의 시간이 기준입니다.

예:

- `약초(15초)` + `붉은꽃잎(30초)` -> 결과는 성장포션
- 작업시간 기준은 `30초`

### 제작 시간

장비나 재료 제작도 결과물 시간이 아니라, 조합에 넣은 2개 재료 시간으로 계산됩니다.

예:

- `구리조각(30초)` + `은광석(60초)`
- 기준시간은 `60초`

### 강화 시간

같은 아이템을 합쳐 강화할 때는 두 재료가 같으므로 계산이 더 단순합니다.

예:

- `은광석+1` 두 개를 강화
- 각 재료 시간은 `60초 × 2 = 120초`
- 결과 기준시간도 `120초`

## 13. 주의할 점

- `brewDuration`이라는 이름 때문에 양조 전용처럼 보이지만 실제로는 공용 기준시간입니다.
- 결과물 이름만 보고 시간을 추정하면 안 되고, 넣은 재료의 기본시간을 기준으로 계산해야 합니다.
- 일부 테스트 아이템과 기록 조각처럼 `0초` 또는 매우 짧은 예외 항목이 있습니다.
- 실제 플레이 중 최종 시간은 스킬, 효과, 자동화 상태에 따라 더 줄어들 수 있습니다.
